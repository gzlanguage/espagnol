Términos de Uso

Última actualización: 3/9/2025.

Bienvenido a GZ. Al acceder a nuestra aplicación web y utilizar nuestros servicios, aceptas estos términos de uso. Por favor, léelos cuidadosamente. Si no estás de acuerdo con alguna de las cláusulas, te recomendamos que no uses nuestra aplicación.

1. Propósito de la Aplicación

G-Z es una plataforma gratuita que permite a los usuarios de todo el mundo enviar información demográfica (fecha de nacimiento, país de nacimiento, género) para generar estadísticas globales sobre nacimientos, y mucho más. Nuestro objetivo es promover una mejor comprensión de las tendencias demográficas y proporcionar datos accesibles para fines educativos, científicos y sociales.

2. Acceso y Uso de Servicios

El uso de la aplicación G-Z está sujeto a las siguientes reglas:

  • Sin Costo: Todas las funciones y datos de la aplicación son gratuitos y accesibles para todos. No se debe monetizar ni vender ninguna información, directa o indirectamente.
  • Uso Personal o Educativo: La aplicación está diseñada para el uso personal, educativo o de investigación no comercial. Cualquier intento de utilizar nuestros datos o servicios para fines comerciales o lucrativos está estrictamente prohibido.

3. Restricciones de Uso

Al usar GZ, aceptas:

  1. No utilizar nuestros datos o servicios para actividades fraudulentas, ilegales o poco éticas.
  2. No reproducir, distribuir, vender o explotar los datos generados por nuestra aplicación para fines comerciales o lucrativos.
  3. No acceder, manipular o intentar interferir con los sistemas o servidores de G-Z de manera no autorizada.
  4. Respetar el objetivo de transparencia y acceso libre de la aplicación al no reclamar derechos de propiedad sobre los datos generados.

4. Derechos de Propiedad Intelectual

Todo el contenido, código, diseño o funcionalidad de la aplicación G-Z permanece como propiedad exclusiva de sus creadores. No adquieres derechos de propiedad al utilizar la aplicación. Se te permite consultar, descargar y utilizar las estadísticas solo de acuerdo con las reglas mencionadas anteriormente.

5. Protección de Datos y Anonimato

GZ se compromete a proteger la información que envías de acuerdo con nuestra Política de Privacidad.

  • Todos los datos se agregan y anonimizan de inmediato.

  • Cualquier intento de desanonimizar o utilizar los datos para identificar a un individuo está estrictamente prohibido y puede resultar en sanciones legales.
